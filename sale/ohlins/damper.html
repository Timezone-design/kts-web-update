OHLINS 車高調KIT
メーカーHPはコチラ
高精度・低フリクションを追求したOHLINS製車高調KITです。
ストリート、ワインディング、サーキットと走行環境を選ばず、乗り心地と走行性能を高い次元で両立する車高調KITです。
【サブバルブ】
DFV：
高すぎる傾向になる中速から高速域にかけての減衰力を適切にカット。低速域はしなやかさと運動性を両立した特性のまま、乗り心地にも優れた理想的な減衰力特性を実現。
PCV：
メインピストンに加え並列に動作するサブピストンを設けた構造。圧縮時におけるシリンダー内のオイルの流れをコントロールし、理想的な減衰特性を実現。
【ショックアブソーバータイプ】
Type N ： ノーマル形状モデル
Type NS ： ノーマル形状ショートモデル
Type HA ： ネジ式車高調モデル
Type HAL ： ネジ式車高・全長調整モデル
Type PB ： ネジ式車高調整別タンクモデル
