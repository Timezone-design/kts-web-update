【Racing MONO6】
エンドレス初の鍛造モノブロックキャリパー
数千トンもの圧力によって鍛造成型されたモノブロックキャリパー「RacingMONO6」。
構想から4年、試作完成から1年、テスト・レースへの実戦投入を繰り返し、幾度となく 仕様変更を行うことで、戦い抜くために必要にして十分な、軽量・高剛性ボディを手に 入れました。そして、難易度の高いピストンシリンダーの加工は、高度な特殊技術を 用いることでクリア。完成したボディは超硬質アルマイト処理で仕上げました。 鍛造にこだわり続けるENDLESSの「終わりなき挑戦」が生み出した技術の結晶です。
【主な仕様】
重量： 2,930g
寸法： D177mm×H76mm×W295mm
対応ローター径： φ355～φ390
対応ホイールサイズ： 18インチ～
専用ブレーキパッド： TYPE R / MX72 / MX72 PLUS
【Racing MONO4】
性能にこだわり生み出した4ピストンモノブロックキャリパー
数千トンもの圧力によって鍛造成型されたモノブロックキャリパーの4POTキャリパーバージョン。軽量化、耐久性、放熱性等、性能を追及しテストを重ね、何度も細部の形状の見直しをはかり、完成しました。また、ホイールは15インチから装着が可能となり、さらに装着可能ホイールが増えました。そして、難易度の高いピストンシリンダーの加工は、高度な特殊技術を用いることでクリア。完成したボディは超硬質アルマイト処理で仕上げました。鍛造にこだわり続けるENDLESSの「終わりなき挑戦」が生み出した技術の結晶です。

【12PISTON】
世界初！12ピストンキャリパー
世界初となる「12 PISTON CALIPER」は、超小型6ポットキャリパーの人気モデル「micro six」が持つ3ピストン・1パッドスタイルをベースに、１つのキャリパーに専用設計の4枚のブレーキパッドを使用した一体型ツインキャリパー。これまでの常識を覆す、高い制動性能を発揮します。 鍛造アルミ製とすることで、大型ながら軽量＆高剛性を実現。また、他モデル同様に異径ピストンを採用することでブレーキパッドの偏摩耗を防ぎ、コントロール性も抜群です。
