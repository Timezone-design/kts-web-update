ENDLESS ブレーキローター BASIC取付SETメニュー
メーカーHPはコチラ
スリットレスモデル 純正交換用1ピースローター
ブレーキローター「BASIC」は、スリットレスモデルの1ピースです。ブレーキパッドの性能をもっとも発揮できるFC鋳鉄に特殊熱処理を行うことで、素材の組織を安定させ、ハード走行によるローターの熱膨張、収縮による細かいひび割れ（ヒートクラック）を大幅に減少。ロングライフの使用に対応できます。
【主な仕様】
・材質：FC鋳鉄（ねずみ鋳鉄）
・形状：1ピースタイプ
・特殊熱処理加工済み
【セット内容】
・ENDLESS ブレーキローター BASIC
・取付工賃
